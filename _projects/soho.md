---
title: "Soho E-Commerce Website"
---

An e-commerce shopping website developed as part of my Web Programming 1 course, focusing on both functionality and visual design. The project was built using a full-stack approach, combining front-end and back-end technologies to create a complete online shopping experience.
<br>

**Technologies Used:**

- HTML, CSS, JavaScript<br>
- PHP<br>
- MySQL (phpMyAdmin)<br>

**Key Features:**<br>
- Clean and modern user interface<br>
- Multiple product categories for easy browsing<br>
- Dynamic product display<br>
- Fully functional shopping cart system<br>
- Interactive frontend using JavaScript<br>
- Backend integration with PHP and MySQL<br>
<br>
The Soho website provides a smooth and intuitive shopping experience, allowing users to browse products, view details, and manage items in a shopping cart. The interface is designed to be visually appealing while maintaining usability and clarity.
<br>
On the backend, the system handles product data, user interactions, and cart functionality using PHP and MySQL, ensuring that data is stored, retrieved, and updated efficiently.
<br>
The goal of this project was to build a complete full-stack web application, demonstrating my ability to integrate front-end design with back-end logic, database management, and user interaction handling.
