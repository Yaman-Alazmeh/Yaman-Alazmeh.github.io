A third-person DeadByDaylight theme game built with Unreal Engine 4.  
Features:
- Fixing geneartor
- Openening the gate
- Escapse the killer


This project was developed to develop similar systems to the famous survival game Deadbydalylight featuring a survivor and killer chasing and opening a gate and fixing a generator.
